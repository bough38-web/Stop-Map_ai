Challenge: Status Classification Logic

Raw licensing data contains vague status codes. Developed a custom parsing logic to categorize businesses into actionable groups: "Active Opportunity" (Construction/Closed) vs "Dead Leads". 


def classify_business_status(row):
    # Identify high-value 'Construction' targets
    if '신축' in row['license_type'] or row['area'] > 1000:
        return 'CONSTRUCTION_SITE'  # High Priority
    
    # Identify 'Closed' businesses (Potential for new opening)
    if row['state'] == '폐업':
        return 'CLOSED_VACANCY'     # Medium Priority
        
    return 'OPERATING'
