# Field Sales Assistant (영업기회 현장 비서)

이 프로젝트는 영업 현장에서 유용한 정보를 지도와 리스트 형태로 제공하여 영업 활동을 지원하는 웹/모바일 애플리케이션입니다.

## 🚀 주요 기능

*   **📊 지사별/담당자별 영업 현황 대시보드**: 실시간 영업 성과를 한눈에 파악할 수 있습니다.
*   **🗺️ 위치 기반 영업 기회 지도**: 내 주변의 영업 기회(인허가/폐업)를 지도에서 직관적으로 확인합니다.
*   **📱 모바일 최적화 리스트**: 현장에서 바로 사용할 수 있는 반응형 리스트 뷰를 제공합니다.
*   **🔍 강력한 필터링**: 날짜, 영업 상태, 키워드, 연락처 유무 등 다양한 조건으로 데이터를 정밀하게 조회합니다.

## 🛠️ 설치 및 실행 방법

1.  **필수 요구사항**
    *   Python 3.9 이상
    *   Streamlit

2.  **설치**
    ```bash
    pip install -r requirements.txt
    ```

3.  **실행**
    ```bash
    streamlit run app.py
    ```

## 📂 프로젝트 구조

*   `app.py`: 메인 애플리케이션 코드
*   `utils.py`: 데이터 처리 및 유틸리티 함수
*   `data/`: 데이터 파일 저장소 (Git 제외됨)
